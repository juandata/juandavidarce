Bienvenidos a mi página web personal

En este espacio web podrán ver algunos de mis diseños. Soy Front-end y Back-end developer con experiencia en diseño de páginas web basadas en HTML5, CSS3, PHP, MYSQL entre otros.

Diseños optimizados

Mis diseños están optimizados para que puedan ser observados desde cualquier dispositivo móvil. Una de las grandes ventajas de trabajar directamente con el código del diseño es poder implementar reglas que adapten el contenido a las diferentes pantallas del mundo digital.

Compatibilidad con los navegadores

Mis diseños están optimizados para verse bien en la gran mayoría de navegadores gracias a reglas específicas que indican como se deben convertir las etiquetas HTML dependiendo del navegador y su versión.
